# NemoClaw on DGX Spark

The DGX Spark guide has moved to the official documentation site. DGX Spark needs no platform-specific pre-setup — Docker is pre-installed — so the standard [Quickstart](https://docs.nvidia.com/nemoclaw/latest/get-started/quickstart.html) works directly. The end-to-end walkthrough, including local inference with Ollama on the Spark GPU, now lives in the **Tutorials** section.

- **DGX Spark tutorial:** <https://docs.nvidia.com/nemoclaw/latest/get-started/tutorials/dgx-spark.html>
- **Platform Setup (hub):** <https://docs.nvidia.com/nemoclaw/latest/get-started/platform-setup/>
- **Quickstart:** <https://docs.nvidia.com/nemoclaw/latest/get-started/quickstart.html>

If you are browsing the repository on GitHub, the tutorial source is [`docs/get-started/tutorials/dgx-spark.md`](docs/get-started/tutorials/dgx-spark.md).
